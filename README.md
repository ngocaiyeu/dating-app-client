# datting_social

Social networking apps. FrontEnd written in Flutter. BackEnd written in golang.(Is in the process of finishing)

Include function
- SignUp, SignIn
- JWT
- Show list of users
- Post an status
- Show list of posts
- Profile
- Edit Profile
- Like Post (In progress)
- Follow (In progress)
- Send message (In progress)
- Show list message (In progress)
